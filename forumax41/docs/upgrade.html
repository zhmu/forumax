ForuMAX upgrade
This document will describe how to upgrade ForuMAX.

Upgrading ForuMAX
Upgrading ForuMAX is much easier than installing it. All you have to do is copy all files, except forum_options.pl from the cgi-bin directory to the cgi-bin directory of the forum. That's it.

Notice If you are upgrading from any version before 4.0 to 4.0, you will also need to do the stuff below:

1) Create an empty file called groups in the internal ForuMAX directory. It must have the same owner and permissions as accounts has.
2) Edit forum_options.pl, and add the following:

$GROUPDB_FILE=qq~Complete directory and filename to your groups file~;
$GROUPDB_LOCKFILE=qq~Complete directory and filename to your groups file, but add .LOCK to it~;

When you are in the CP, immediately submit the options, and it'll be fine.

Notice If you are upgrading from 4.0 to 4.1, you will also need to check the Rebuild moderator list checkbox in the Control Panel. This only needs to be done once.

Good luck, and please enjoy ForuMAX!
