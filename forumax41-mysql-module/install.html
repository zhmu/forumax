ForuMAX installation
This document will describe how to install the MySQL user and group module for ForuMAX 4.0.

Things you need In order to use the MySQL module, you'll need:

  • Perl 5 or above
  • MySQL installed
  • A MySQL database, where you can create tables in
  • The Perl DBI module, and the MySQL DBD module

Most likely, the latter is a problem. You can find the modules using a search engine, at http://www.perl.com and most likely at http://www.mysql.org.

Once you've done that, copy conv_db.cgi and user_db_mysql.pl to your ForuMAX directory, and run conv_db.cgi from your web browser. When it has finished without errors, you can delete these two files, they will not be used anymore.

Good luck, and please enjoy ForuMAX!
